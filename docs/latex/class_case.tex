\doxysubsection{Detailed Description}
Represents a case in the Deal or No Deal game. 

A \doxylink{class_case}{Case} object contains an amount of money, a sprite for display, and text representing the amount. The case can be selected, deselected, or eliminated, and has a visual representation in the game window.
