\doxysubsection{Detailed Description}
Represents a \doxylink{class_banker}{Banker} who supports the player with generous offers. 

The \doxylink{class_helper_banker}{Helper\+Banker} makes offers that scale positively with the remaining case amounts, often erring on the player\textquotesingle{}s side.
