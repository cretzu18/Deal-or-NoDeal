\doxysubsection{Detailed Description}
Handles the game settings such as volume levels and provides a menu interface. 

The {\ttfamily \doxylink{class_settings}{Settings}} class allows the user to adjust game settings such as menu and game volume levels. It provides methods to draw the settings menu, update the settings based on user input, and access the volume levels for the menu and the game.
