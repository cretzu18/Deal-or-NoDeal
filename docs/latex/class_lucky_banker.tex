\doxysubsection{Detailed Description}
Represents a \doxylink{class_banker}{Banker} whose offers are influenced by random chance. 

The \doxylink{class_lucky_banker}{Lucky\+Banker} applies a random multiplier to the average of the remaining case amounts, resulting in unpredictable offers.
