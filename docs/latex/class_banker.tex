\doxysubsection{Detailed Description}
Base class representing a \doxylink{class_banker}{Banker} in the game. 

The \doxylink{class_banker}{Banker} class handles the calculation of offers, rendering of accept/reject buttons, and maintains the history of offers. It is designed to be extended by derived classes that provide specific offer calculation strategies.
